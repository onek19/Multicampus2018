### 사용자 계정 만들기
1. 계정 만들기
  - create user 이름 identified by 비밀번호;


2. 만든 계정 lock 풀기
  - alter user 계정명 account unlock;
  - create로 만든 것들은 alter 로 수정
  - Not connected 라 나올 경우 > 수정 권한이 없기 때문
    -  conn [= connected] system/oracle; 을 통해 권한 부여 후 다시 alter~ 쓰기
    - -> (결과) User altered 출력
